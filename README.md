 # King of the UnderWorld
### Unity2D Project
점프킹에 RPG 요소를 더하여 점프맵 발판 중간중간에 있는 일반몬스터를 처치해가며 목표를 향해 올라가는 게임입니다. 목표지점 전에는 보스몬스터가 있으며 처치 시 아이템을 획득할 수 있습니다. 첫 번째 캐릭터로 게임을 클리어 시 스토리 전개에 따라 두 번째 캐릭터로 게임을 플레이 하게 됩니다.
